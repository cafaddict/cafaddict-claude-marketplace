# cafaddict-claude-marketplace

Claude Code 플러그인 마켓플레이스 — 생산성 향상을 위한 플러그인 모음.

## 설치

```bash
# 레포 클론
git clone https://github.com/cafaddict/cafaddict-claude-marketplace.git

# 원하는 플러그인 설치
claude plugin add ./cafaddict-claude-marketplace/tf
```

## 플러그인 목록

| 플러그인 | 설명 | 버전 |
|----------|------|------|
| [tf](./tf) | thinkfirst — 체계적 사고 과정을 강제하여 이해의 부채를 방지한다 | 0.1.0 |

각 플러그인의 상세 사용법은 해당 디렉토리의 README.md를 참고하세요.
